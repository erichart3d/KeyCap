First-run setup

one-time — so you never touch PowerShell again

1 · Desktop shortcut

Creates KeyCap.lnk on your desktop that starts overlay_server.py and opens this editor in your browser. Double-click whenever you want to record.

2 · Launch on login (optional)

Add KeyCap to the Windows Startup folder. Server will boot quietly in the tray.

3 · Manual command

If you'd rather run it yourself:

cd C:\Projects\KeyDisplay python overlay_server.py
